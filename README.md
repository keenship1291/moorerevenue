# MooreRevenue Website

This is the official website for **MooreRevenue**, an AI automation agency specializing in voice agents, websites, and workflow automation for service businesses.

## Features
- **Modern Dark Theme**: Premium design with emerald green accents.
- **Responsive Layout**: Fully functional on mobile and desktop.
- **EmailJS Integration**: Contact form sends emails directly without a backend.
- **No-Code Friendly**: Easy to customize and deploy.

## Setup
1. Open `index.html` to edit content.
2. Update `script.js` with your EmailJS credentials.
3. Deploy to Netlify, Vercel, or GitHub Pages.

## Technologies
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (Vanilla)
- EmailJS SDK
